## 功能增强计划

### Phase 1: 状态信息显示 🎯 (推荐优先实现)

#### 1.1 悬浮信息面板
**优先级：⭐⭐⭐⭐⭐**

显示当前 Claude Code 的状态信息：
- 当前状态（thinking/coding/reading...）
- 正在使用的工具名称
- 工具参数摘要（文件名、命令等）
- 状态持续时间

**实现方案：**
```typescript
// src/components/StatusPanel.vue
- 浮动在 Live2D 旁边
- 自动隐藏/显示
- 支持拖拽
- 半透明背景
```

**用户体验：**
```
┌─────────────────┐
│ 🤔 Thinking     │
│ Duration: 5s    │
└─────────────────┘
     ↓
  [Live2D]
```

#### 1.2 工具详情展示
**优先级：⭐⭐⭐⭐**

点击桌宠显示详细信息：
```typescript
{
  tool: "Edit",
  file: "src/main.ts",
  lines: "10-25",
  timestamp: "2分钟前"
}
```

#### 1.3 操作历史
**优先级：⭐⭐⭐**

保存最近的操作历史：
- 最近 10 次操作
- 操作时间戳
- 操作结果（成功/失败）

### Phase 6: Claude Code 深度集成 🔗

#### 6.1 项目信息显示
**优先级：⭐⭐⭐⭐**

从 Claude Code 获取：
- 当前项目名称
- 工作目录
- Git 分支名

**实现方式：**
Hook 脚本传递更多信息：
```bash
# 修改 hook 脚本
PROJECT_NAME=$(basename "$PWD")
GIT_BRANCH=$(git branch --show-current 2>/dev/null)

echo "{
  \"state\": \"$STATE\",
  \"project\": \"$PROJECT_NAME\",
  \"branch\": \"$GIT_BRANCH\",
  \"timestamp\": $TIMESTAMP
}" > ~/.bongo-cat/status.json
```

## 需要你决定的问题 🤔

1. **UI 风格**
   - 最小化（只显示必要信息）
   - 丰富（显示详细统计和图表）

2. **通知方式**
   - 系统通知（macOS Notification Center）
   - 应用内通知（Toast）
   - 两者都支持

3. **数据存储**
   - 本地 JSON 文件
   - SQLite 数据库（用于历史记录）

4. **隐私设置**
   - 是否记录工具参数详情
   - 是否保存操作历史
   - 数据保留时长
